## 使用方法

### 打开演示文稿
1. 在浏览器中打开 `dora_presentation.html` (中文版) 或 `dora_presentation_en.html` (英文版)
2. 建议使用 Chrome、Firefox 或 Edge 浏览器以获得最佳效果

### 导航控制
- **鼠标操作**: 点击底部的导航按钮切换幻灯片
- **键盘快捷键**:
  - `←` `↑` - 上一页
  - `→` `↓` `空格` - 下一页
  - `F` - 全屏模式
  - `ESC` - 退出全屏
  - `Home` - 第一页
  - `End` - 最后一页
  - `1-8` - 直接跳转到对应页面

### 触摸支持
- **滑动操作**: 在触摸屏设备上左右滑动切换幻灯片
- **触摸导航**: 点击右侧的页面指示器直接跳转

### 高级功能
- **全屏模式**: 点击"全屏"按钮或按 F 键进入全屏演示
- **打印功能**: 浏览器打印功能可将演示文稿导出为 PDF
- **自动播放**: 可通过开发者工具启用自动播放功能

## 转换为 PowerPoint

如需将此 HTML 演示文稿转换为 PowerPoint 格式：

### 方法一: 直接导入
1. 打开 PowerPoint
2. 选择"插入" > "屏幕录制"
3. 录制演示文稿的每一页
4. 将录制的视频插入到 PowerPoint 幻灯片中

### 方法二: 截图导入
1. 全屏打开演示文稿
2. 使用截图工具截取每一页
3. 将截图导入到 PowerPoint 幻灯片中
4. 在 PowerPoint 中重新添加动画效果

### 方法三: 在线转换工具
使用在线 HTML 转 PPT 服务，如:
- Zamzar
- Online-Convert
- Convertio

## 自定义和修改

### 修改内容
- 直接编辑 HTML 文件中的文本内容
- 保持 HTML 结构不变以确保正常显示

### 修改样式
- 编辑 `<style>` 标签中的 CSS 代码
- 可调整颜色、字体、布局等

### 添加幻灯片
1. 复制现有的 `<div class="slide">` 结构
2. 修改 ID 和内容
3. 更新 JavaScript 中的 `totalSlides` 变量
4. 添加对应的页面指示器

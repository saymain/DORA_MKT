Quick Start

Build Your First Dataflow Application in Minutes

1. Install

pip install dora-rs-cli
# or cargo install dora-cli

2. Run Examples

dora build yolo.yml --uv
dora run yolo.yml --uv

3. Build Dataflows

import dora
with dora.Builder() as builder:
  camera = builder.node("opencv-video-capture")
  yolo = builder.node("dora-yolo")
  camera.output("image").to(yolo.input("image"))
